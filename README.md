# Rick Segal portfolio

### Live site: https://www.rick-segal.com/

A modern, minimalist portfolio I built for a friend who works as an Executive Producer. Built with [Next.js](https://nextjs.org) (the website) and [Sanity](https://www.sanity.io/) (the CMS). Deployed on [Netlify](https://www.netlify.com/).

This is actually the third version of the project. The project itself has largely remained the same, but the underlying technology has modernized over time. Previous verions include:

1. AngularJS in 2017 ([repo link](https://github.com/johnnycopes/rick-portfolio-old))
2. Gatsby + Netlify CMS in 2019 ([repo link](https://github.com/johnnycopes/rick-portfolio-v2))

#### Goals of this project:

- Learn more about Next.js and SSR
- Learn how to use and configure Sanity
- Learn more about Claude and AI coding by leveraging it for the migration

#### Local installation:

#### Website

1. Clone this repo
1. Run `npm install` to install all local dependencies
1. Run `npm run dev` to run the development server
1. Open [http://localhost:3000](http://localhost:3000) in your browser to see the website

#### CMS

1. Clone the [Sanity repo](https://github.com/johnnycopes/rick-portfolio-studio)
1. Run `npm install` to install all local dependencies
1. Run `npm run dev` to run the development server
1. Open [http://localhost:3333](http://localhost:3333) in your browser to see the Sanity Studio CMS
